Clustering and Merge

Clustering and Merge

Description

How to find it?

Clustering and merging usually are using to fix the classify and merge same data which has been identified into different data improperly.

In this dataset, this mainly happened in the attribute "Project Name". So, using the text facet to analysis the situation of this attribute. Then using the function "Cluster" to address this probelm.

How to deal with it?

Using the "cluster" to analysis and identify which items are same, then click them and merge them under the same "Project Name".

Data Cleaning
Misspelling

Misspelling

Description

How to find it?

After clustering and merging the same prject, I noticed that some project names are highly identical. For exsample, the project "Department of Agriculture" and "Department of Agraculture". After investigating these two names, these is no department named "Department of Agraculture" in US government. So, it's clear that it's a misspelling error.

How to deal with it?

Find the misspelling data in the Facet Box, then click the edit button. After that you can correct the mistake, and it will be merged into the same group.

Duplicate Records

Description

How to find it?

How to deal with it?

Mark all reduplicate records,then using ALL -- Edit Rows -- remove all markd rows to remove reduplicate records.
