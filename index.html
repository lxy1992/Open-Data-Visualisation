Clustering and Merge

Clustering and Merge

Description

How to find it?

Clustering and merging usually are using to fix the classify and merge same data which has been identified into different data improperly.

In this dataset, this mainly happened in the attribute "Project Name". So, using the text facet to analysis the situation of this attribute. Then using the function "Cluster" to address this probelm.

How to deal with it?

Using the "cluster" to analysis and identify which items are same, then click them and merge them under the same "Project Name".

Misspelling

Misspelling

Description

How to find it?

After clustering and merging the same prject, I noticed that some project names are highly identical. For exsample, the project "Department of Agriculture" and "Department of Agraculture". After investigating these two names, these is no department named "Department of Agraculture" in US government. So, it's clear that it's a misspelling error.

How to deal with it?

Find the misspelling data in the Facet Box, then click the edit button. After that you can correct the mistake, and it will be merged into the same group.

Combine Attributes

Combine Attributes

Description

How to find it?

There are two attributes which are describe the "Updated Date". Merge the "Updated Date" and "Updated Time" into one column.

How to deal with it?

Choose the Add column base on column of "Updated Date". Then type the code cells['Updated Date'].value + ' ' + cells['Updated Time'].value into the Expression form to merge these two columns.

Data Cleaning
Inconsistent Formats

Inconsistent Formats

Description

How to find it?

1. Extract Symbol Format: The data in the column of "Lifecycle Cost" has some number with dollar sign "($m)". It will cause some problem in visualisaion. Using Numberic Facet and choose Non-numberic can find these wrong data.

2. Number Format: Some numbers have extra comma separating like '1,000'.

How to deal with it?

1. Using Edit Cells-->Transform, type in the Expression value.replace('($m)',"") to remove all doller sign. Then use Edit Cells --> Common Transforms--> To number to rule the format.

2. Using Numberic Facet on "Project ID", then choose the Non-numeric to change it by using Expression value.replace(",","") to remove all "," in Project ID.
